Assessment Structure

Sections, Duration, Difficulty

Approximate breakdown to guide practice. Capgemini leans on gamified cognitive rounds instead of traditional aptitude/verbal batteries.

Gamified Cognitive (Attention + Logic)

Duration: ~20–25 mins • Difficulty: Moderate

  • Pattern spotting and matrix completion
  • Rule switching and selective attention
  • Speed/accuracy tradeoffs with simple stimuli

Communication & Reading

Duration: ~15–20 mins • Difficulty: Easy–Moderate

  • Reading comprehension
  • Sentence improvement, grammar basics
  • Vocabulary in context

Technical (CS Fundamentals)

Duration: ~20–25 mins • Difficulty: Moderate

  • OOP, DBMS joins/keys, OS basics, CN fundamentals
  • Simple DSA: arrays/strings, searching/sorting logic
  • Basic debugging / output prediction

Coding (Basic to Medium)

Duration: ~20–30 mins • Difficulty: Moderate

  • Loops, conditions, functions with clear I/O
  • Array/string manipulation and edge cases
  • Emphasis on readability and correctness

Pro Tips

Do & Don’t

Stay precise, manage time, and practice ethically.

Do

  • Practice daily with a timer
  • Read questions fully before answering
  • Prepare 2 strong project walkthroughs
  • Write clean, readable code with edge cases

Don’t

  • Guess blindly; eliminate and skip if unsure
  • Ignore time; budget per section
  • Panic on hard items; move and revisit
  • Use leaked materials; stay ethical
